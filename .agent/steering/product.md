# Product Overview

This is a personal portfolio website built as a single-page application (SPA). The site features:

- Modern, responsive design with dark/light theme support
- Interactive cosmic/space-themed visual effects (stars, meteors, floating animations)
- Portfolio sections: Hero, About, Skills, Projects, Contact
- Integrated chatbot widget for visitor interaction
- Toast notifications for user feedback

The application emphasizes visual appeal with custom animations and a polished, professional presentation suitable for showcasing development work and personal branding.
